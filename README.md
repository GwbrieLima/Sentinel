# Sentinel
Sistema de monitoramento climático desenvolvido em Python.

## Funcionalidades
Consulta de clima em qualquer cidade
Alertas automáticos
Previsão para os próximos dias
Classificação de risco
Histórico de consultas

## Tecnologias
Python
Requests
WTTR.in API

## Como executar

```bash
pip install requests
python main.py
```
